# NOREEN Educational Funding Platform

## Project Overview

NOREEN is a web-based platform designed to improve access to educational funding and academic guidance by connecting beneficiaries, investors, consulting offices, and administrators within a unified digital environment.

The platform aims to simplify funding processes, improve communication among stakeholders, and support data-driven decision-making.

## Problem Statement

Students and beneficiaries often face challenges in finding suitable funding opportunities and obtaining reliable educational guidance. Existing solutions are fragmented and lack effective coordination between stakeholders.

## Solution

NOREEN provides an integrated platform that:

- Connects beneficiaries with funding opportunities.
- Supports investors in evaluating and managing funding requests.
- Enables consulting offices to provide academic support services.
- Centralizes information and communication within a single system.

2. System Design
3. Implementation
4. Testing
